

## 1. 身份证归属地查询

请求路径：http://42.192.74.35:8080/query/getLocationByIdCard

请求方法：POST

请求参数：
```json
{
  "idCard":"211224********7133"
}
```
返回信息：

```json
{
  "code":"000000",
  "data":"辽宁省铁岭市昌图县"
}
```

## 4.保存留言

请求路径：http://42.192.74.35:8080/message/saveMsg

请求方法：POST

参数：

```json
{
  "openId":"1234567890",
  "userName":"wechat_name",
  "messageTitle":"who are you",
  "message":"can you tell me who are you ?"
}
```



返回信息：

```json
true/false
```
